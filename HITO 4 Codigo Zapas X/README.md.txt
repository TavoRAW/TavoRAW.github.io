# Zapas X

 Descripción General

**Zapas X** es una tienda web ficticia de zapatillas de edición limitada. El sitio simula un e-commerce moderno, responsivo y visualmente atractivo que permite a los usuarios explorar productos, agregar zapatillas a un carrito de compras y simular una compra.

Este proyecto fue desarrollado con fines educativos para demostrar el uso de tecnologías web modernas y lógica básica de e-commerce sin backend.

 Qué problema resuelve o qué simula

Simula una experiencia básica de compra online con:
- Interacción dinámica (agregar al carrito, confirmar compra).
- Diseño adaptativo para celular, tablet y escritorio.
- Experiencia de usuario mejorada con animaciones e íconos visuales.

---

## Tecnologías Utilizadas

- **HTML5** – Estructura del sitio.
- **CSS3** – Estilos visuales personalizados.
- **JavaScript (vanilla)** – Lógica del carrito de compras y carrusel.
- **Google Fonts** – Tipografía.
- **Boxicons** – Íconos de carrito y menú.
- **IA de OpenAI (ChatGPT)** – Apoyo en generación y mejora de código y diseño.

---

## Funcionalidades

-  Carrito de compras funcional (agregar, mostrar total, limpiar después de compra).
-  Carrusel dinámico con navegación por botones.
-  Confirmación de compra con mensaje visual animado.
-  Interfaz completamente responsiva (desktop, tablet, móvil).
-  Barra de navegación con búsqueda (no funcional por ahora).
